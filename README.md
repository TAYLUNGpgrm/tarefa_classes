# Projeto de Práticas em Java - Matemática e Estruturas

Este repositório contém exercícios práticos desenvolvidos para consolidar conceitos fundamentais da linguagem Java, com foco em lógica de programação, orientação a objetos e boas práticas de documentação.

## Tecnologias Utilizadas
* **Java 17** (ou a versão que estiveres a usar)
* **IntelliJ IDEA**
* **Javadoc** (para documentação de código)
* **Git/GitHub**

##  Estrutura do Módulo `srcMath`
O projeto está organizado da seguinte forma dentro da pasta principal:

- **`Matematica.Math1`**: 
    - `Tabuada.java`: Implementação de lógica de repetição para gerar tabuadas dinâmicas.
    - `Atribuicao.java`: Classe para demonstração de modificadores de acesso (`public`, `private`, `protected` e *default*).
- **`Matematica.Resultado`**:
    - `PrintResult.java`: Classe que utiliza herança (`extends`) para interagir com os membros da classe `Atribuicao` e exibir resultados.

## Conceitos Aplicados
1. **Documentação Javadoc**: Utilização de tags como `@author`, `@param` e `@see` para gerar documentação técnica padronizada.
2. **Modificadores de Acesso**: Exploração da visibilidade de atributos e métodos entre diferentes pacotes.
3. **Lógica de Controlo**: Uso de estruturas de repetição (`while` e `for`) e entrada de dados via `Scanner`.
4. **Herança**: Demonstração de como uma subclasse pode aceder a atributos protegidos da superclasse.
